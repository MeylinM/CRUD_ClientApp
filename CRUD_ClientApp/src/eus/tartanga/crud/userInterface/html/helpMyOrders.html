My Orders: Help

General Description

This window allows the user to review their purchase history within the application. Using the available controls, the user can filter orders to find specific details and view รง the information of each purchase. Additionally, the window offers the option to generate a report containing a record of all completed purchases.

Operations:

Viewing past purchases

The order history displays a list of all completed purchases, including details such as order date, purchased products, quantity, and total price.

Filtering purchases

You can filter your past purchases using the available filters:

  • By artist: Use the dropdown menu to select an artist and display only the purchases related to their products.
  • By date range: Use the date pickers to filter purchases within a specific time frame.
Generating a report

To generate a purchase history report, press the "Print" button. This will create a document containing a record of all completed purchases.

Help

Pressing the Help button will display this help document.
